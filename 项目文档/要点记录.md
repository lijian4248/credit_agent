以下是具体的实现步骤：

### 1. 定义分析报告的数据结构
首先，我们需要明确大模型最终输出的报告长什么样。这将指导我们后续的提示词编写和代码逻辑。

*   **目标**：确定报告包含的字段。
*   **建议结构**：
    *   `risk_level` (String): 风险等级（高/中/低）。
    *   `risk_points` (List[String]): 具体的风险点列表（如“近24个月逾期6次”）。
    *   `suggestions` (List[String]): 针对性的建议列表（如“建议降低授信额度”）。
    *   `analysis_details` (String): 详细的分析过程或理由。

### 2. 设计提示词模板
这是核心环节。我们需要编写一个专门的提示词，让 LLM 理解它是一位风控专家，并按照我们的规则进行分析。

*   **位置**：在 `config/prompt_config.py` 中新增。
*   **关键要素**：
    *   **角色设定**：你是一位资深信贷风控专家。
    *   **任务描述**：依据提供的规则库，分析用户资料，输出风险评估报告。
    *   **输入数据**：
        *   `{rules}`：从 `data/knowledge/credit_rules.md` 和 `data/knowledge/信贷风控.txt` 加载的规则文本。
        *   `{custom_rules}`：用户自定义的规则文本。
        *   `{user_profile}`：用户上传的资料解析后的文本。
    *   **输出要求**：必须输出 JSON 格式，并符合步骤 1 定义的结构。
*   **示例代码**：
    ```python
    # config/prompt_config.py
    ANALYSIS_PROMPT = """
    你是一位资深信贷风控专家。请依据以下规则，对用户资料进行分析，并输出风险评估报告。
    
    ### 规则库 ###
    {rules}
    
    ### 自定义规则 ###
    {custom_rules}
    
    ### 用户资料 ###
    {user_profile}
    
    ### 输出要求 ###
    请严格以 JSON 格式输出，包含以下字段：
    - risk_level: 风险等级（高/中/低）
    - risk_points: 风险点列表
    - suggestions: 建议列表
    - analysis_details: 详细分析
    
    不要输出任何 JSON 之外的文字。
    """
    ```

### 3. 实现规则加载逻辑
我们需要一个函数来合并静态规则和自定义规则，以便注入到提示词中。

*   **位置**：建议在 `rag/llm_chain.py` 中新增，或创建 `utils/rules_loader.py`。
*   **逻辑**：
    1.  读取 `data/knowledge/credit_rules.md` 和 `data/knowledge/信贷风控.txt` 的内容。
    2.  读取自定义规则（初期可以是硬编码的字符串，或从 `data/custom_rules.json` 读取）。
    3.  将它们拼接成一个长字符串，中间用换行符分隔。
*   **示例代码**：
    ```python
    def load_rules():
        # 读取静态规则
        with open("data/knowledge/credit_rules.md", "r", encoding="utf-8") as f:
            static_rules = f.read()
        # 读取自定义规则（示例）
        custom_rules = "自定义规则：餐饮行业客户，近3个月流水下滑超过20%，标记为高风险。"
        return f"{static_rules}\n\n{custom_rules}"
    ```

### 4. 构建分析链
这是最核心的代码部分，类似于现有的 `build_rag_chain`，但不需要检索器（Retriever），因为我们直接把规则作为上下文传给 LLM。

*   **位置**：在 `rag/llm_chain.py` 中新增 `build_analysis_chain` 函数。
*   **逻辑**：
    1.  使用 `ChatPromptTemplate.from_template` 加载步骤 2 定义的 `ANALYSIS_PROMPT`。
    2.  获取 LLM 实例（复用 `get_llm()`）。
    3.  使用 `StrOutputParser` 解析输出（或者使用 `JsonOutputParser` 以获得更稳定的 JSON）。
    4.  构建链：`PromptTemplate | LLM | OutputParser`。
*   **示例代码**：
    ```python
    from langchain_core.output_parsers import JsonOutputParser
    
    def build_analysis_chain():
        prompt = ChatPromptTemplate.from_template(ANALYSIS_PROMPT)
        llm = get_llm()
        # 使用 JsonOutputParser 确保输出是有效的 JSON
        parser = JsonOutputParser() 
        chain = prompt | llm | parser
        return chain
    ```

### 5. 实现资料解析功能
用户上传的文件（如 PDF、图片）需要转换为文本，才能被 LLM 理解。

*   **位置**：建议创建 `utils/file_parser.py`。
*   **逻辑**：
    1.  根据文件扩展名（.pdf, .docx, .png, .jpg）选择不同的解析器。
    2.  对于 PDF/Word，使用 `PyPDF2` 或 `python-docx` 提取文本。
    3.  对于图片，使用 OCR（如 `PaddleOCR`）提取文字。
    4.  返回提取出的纯文本字符串。
*   **示例代码**：
    ```python
    import PyPDF2
    
    def parse_pdf(file_path):
        text = ""
        with open(file_path, 'rb') as file:
            reader = PyPDF2.PdfReader(file)
            for page in reader.pages:
                text += page.extract_text()
        return text
    ```

### 6. 集成到主程序
最后，我们需要在 `main.py` 中将上述组件串联起来，提供用户交互界面。

*   **位置**：`main.py`。
*   **逻辑**：
    1.  加载规则（调用步骤 3 的函数）。
    2.  初始化分析链（调用步骤 4 的函数）。
    3.  提示用户上传文件。
    4.  解析文件（调用步骤 5 的函数）。
    5.  调用分析链，传入 `rules` 和 `user_profile`。
    6.  接收 LLM 返回的 JSON 报告。
    7.  将报告格式化（如打印为 Markdown）并展示给用户。
*   **示例代码**：
    ```python
    # main.py
    from rag.llm_chain import build_analysis_chain, load_rules
    from utils.file_parser import parse_pdf
    
    # ... 初始化代码 ...

    # 2. 构建分析链
    analysis_chain = build_analysis_chain()

    user_profile = parse_pdf(file_path)
    
    # 4. 执行分析
    report = analysis_chain.invoke({
        "rules": rules,
        "custom_rules": "", # 如果有自定义规则接口，在这里传入
        "user_profile": user_profile
    })
    
    # 5. 展示报告
    print("=== 风险评估报告 ===")
    print(f"风险等级: {report['risk_level']}")
    print(f"风险点: {', '.join(report['risk_points'])}")
    print(f"建议: {', '.join(report['suggestions'])}")
    ```
